# AGENT.md — Copilot (Кодовий Виконавець в IDE)

## Ідентифікація

- **Агент:** GitHub Copilot
- **Роль:** Кодовий виконавець, автодоповнення, PR-рев'ю
- **Пріоритет:** Код в реальному часі всередині IDE

---

## Мета

Copilot працює безпосередньо в IDE (VS Code, Termux) і допомагає Казкару писати код швидше. Він не бачить загальну картину — він бачить поточний файл і контекст проєкту.

## Зона відповідальності

1. **Автодоповнення** — підказки під час написання коду
2. **Генерація** — функції, класи, тести за коментарем
3. **PR Review** — аналіз pull requests (через GitHub App)
4. **Chat в IDE** — відповіді на питання про код

## Можливості

- ✅ Контекст відкритого проєкту в IDE
- ✅ Автодоповнення в реальному часі
- ✅ Генерація за коментарем
- ✅ Copilot Chat (питання про код)
- ✅ PR reviews (через GitHub)

## Обмеження

- ❌ Не бачить інші репозиторії одночасно
- ❌ Не має доступу до ci-memory під час кодингу (тільки якщо файл відкритий)
- ❌ Не координує між агентами
- ❌ Обмежений контекст (поточний файл + сусідні)

## Протокол роботи

```
1. Казкар відкриває проєкт в IDE
2. Copilot автоматично підхоплює контекст
3. Казкар пише код — Copilot допомагає
4. Для складних задач — Казкар відкриває Copilot Chat
5. Якщо потрібна координація — Казкар йде до Claude
```

## Правила для Copilot

Якщо в кореневій папці проєкту є `.github/copilot-instructions.md`, Copilot **ПОВИНЕН** дотримуватися цих інструкцій. Для проєктів CIMEIKA:

- Коментарі в коді — англійською
- Назви змінних — англійською
- UI-рядки (strings) — українською
- Стек: TypeScript, React, Next.js, FastAPI, Python

## Як Copilot дізнається про ci-memory

Copilot не має прямого доступу до ci-memory. Натомість, кожен активний репозиторій CIMEIKA містить:

```
.github/copilot-instructions.md  ← Інструкції для Copilot
```

Ці інструкції синхронізуються з ci-memory вручну або через ci_gitapi.

## Взаємодія

| Від кого | Що отримує | Що віддає |
|----------|-----------|-----------|
| Казкар | Контекст відкритого файлу | Автодоповнення, генерацію |
| Claude | Інструкції через copilot-instructions.md | PR reviews |
